InsureAI Dashboard

A real-time AI Insurance dashboard for gig workers, combining fraud detection, premium simulation, market crash modeling, interactive analytics, and AI chatbot support.

Features

AI Analysis: Evaluate daily worker earnings and risk based on weather and movement.

Fraud Detection: Identify suspicious activity and flag fraudulent workers.

Premium & Payout Simulator: Calculate weekly premiums and potential payouts based on disruption severity.

Market Crash Simulator: Simulate a market crash scenario with real-time worker data.

Interactive Analytics: Visualize worker earnings via Chart.js and locations via Leaflet.js maps.

AI Chatbot: Ask questions, get advice, or run simulations. Supports voice input and multiple languages.

Responsive Design: Works beautifully on desktop and mobile devices.

Setup & Usage

Clone the repository:

git clone https://github.com/your-username/insureai-dashboard.git

Navigate into the project folder:

cd insureai-dashboard

Open index.html in a browser (no server required).

All functionality works out-of-the-box using vanilla JavaScript, Chart.js, and Leaflet.js.

How to Use

Login with your name.

Navigate between Dashboard, Analytics, Fraud Alerts, and AI Chatbot via sidebar.

Run AI Analysis for risk assessment.

Calculate Premiums by entering base earnings and severity.

Simulate Market Crash to test system response.

Chat with AI for advice, payouts, or fraud alerts.

Technologies Used

HTML5 & CSS3 – Responsive layout and modern UI.

JavaScript (Vanilla) – Dashboard logic, AI simulations, fraud detection.

Chart.js – Interactive earnings charts.

Leaflet.js – Worker location maps.

Web Speech API – Voice recognition and speech synthesis for AI chatbot.
